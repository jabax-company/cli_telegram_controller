Features

Características principales

🤖

Claude Code CLI Streaming

Output en tiempo real con stream-json. Contexto persistido entre prompts con --resume automático.

🎙️

Voz con Whisper Local

Transcripción con faster-whisper. Sin API externa. Soporte multiidioma con auto-fallback configurable.

🖼️

Imágenes como Contexto

Adjunta fotos a prompts. Usa <image> para referencias. Cola de imágenes pendientes con historial.

✨

Optimizador de Prompts

Q&A interactivo que refina tu prompt antes de enviarlo: goal, paths, constraints, formato y validación.

🌿

Branch Lock

Bloquea Claude a una rama Git específica. Cambia de repo → error automático. Seguridad de código.

🌐

Deploy Cloudflare Tunnel

Estático, proxy, run o fullstack. URL pública instantánea para tu localhost sin cuenta.

🛡️

Seguridad Multinivel

Blocklist de patrones destructivos, Safe Mode, autorización por user ID y restricción de paths.

⏱️

Watchdog de Inactividad

Auto-shutdown de Claude y túneles tras inactividad configurable. Icono de bandeja Windows.
