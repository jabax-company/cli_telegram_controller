Deploy

Publicación web con Cloudflare

A

Sitio estático

/server

Sirve la carpeta actual como HTML/CSS/JS estático

B

Proxy de servicio

/server proxy 5000

Expone localhost:5000 ya corriendo en una URL pública

C

Arrancar + publicar

/server run 3000 npm run dev

Ejecuta el comando y expone el puerto resultante

D

Fullstack (una URL)

/server fullstack 3000 5000

Front + back bajo un solo dominio. Auto-detecta comandos de arranque si los puertos no están activos
