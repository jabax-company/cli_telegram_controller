Install

Instalación paso a paso

0

Requisitos previos

Python 3.13
uv (gestor)
Claude Code CLI
cloudflared

Rojo = obligatorio · Verde = opcional (para /server)

1

Clonar repositorio

PS> git clone <repo-url>
PS> cd cli_telegram_controller
2

Instalar Python y dependencias

PS> uv python install 3.13
PS> uv python pin 3.13
PS> uv sync
3

Configurar variables de entorno

PS> Copy-Item .env.example .env
# Edita .env con tus valores:
TELEGRAM_TOKEN=123456:ABC...
TELEGRAM_USER_ID=987654321

Obtén el token con @BotFather y tu ID con @userinfobot

4

Autenticar Claude Code CLI

PS> claude auth login

Solo necesitas hacerlo una vez. Las credenciales se guardan localmente.

5

Lanzar el bot

PS> uv run python bot.py

# Verás:
CLI Telegram Controller started.
Authorized user: 987654321

¡Listo! Abre Telegram y envía /start
