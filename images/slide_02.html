¿Qué es?

Tu cerebro de IA
en el móvil

Un bot de Telegram que conecta tu teléfono con Claude Code CLI corriendo en tu máquina local.

Envía prompts desde cualquier lugar. Claude ejecuta, escribe código, navega tu sistema de archivos y despliega proyectos — todo sin salir de Telegram.

🤖

Interfaz remota para Claude Code

Lanza prompts, lee output en streaming y mantén el contexto entre sesiones con --resume automático.

📁

Gestión completa del sistema de archivos

Navega carpetas con botones inline, guarda proyectos favoritos, crea directorios y cambia de directorio base.

🌐

Publicación web instantánea

Un comando y tu proyecto local tiene URL pública via Cloudflare Tunnel, sin cuenta ni configuración.

🔒

Privacidad por diseño

Todo corre en tu máquina. Solo tú (tu Telegram user ID) puede operar el bot. Sin servidores intermedios.
